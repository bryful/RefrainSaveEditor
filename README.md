と言ってもチートするというよりデータがおかしくなったセーブデータを修復するために作りました。



# RefrainCTE
同梱されているRefrainCTEはCheat EngineのCTファイルを作成するものです。  
ルフランは実行する度に内部アドレス位置が変わってしまうので、そのたびに走査するのが面倒なので作ったものです。  
一番走査が楽なリーンフォースのアドレス位置だけ調べて、その他の必要な項目のアドレスを計算します。
ExportでCTファイルを出力です。
